# Câu lệnh điều kiện trong python
---
## IF đơn
__Cú pháp:__
```python
if bieu_thuc:
   cac_lenh
```
VD:
```python
var1 = 100
if var1:
   print "1 - Nhan mot gia tri true"
   print var1

var2 = 0
if var2:
   print "2 - Nhan mot gia tri true"
   print var2
print "Good bye!"
```
## IF đầy đủ
__Cú pháp__
```python
if bieu_thuc:
   cac_lenh
else:
   cac_lenh
```
## ELIF
__Cú pháp__
```python
if bieu_thuc1:
   cac_lenh
elif bieu_thuc2:
   cac_lenh
elif bieu_thuc3:
   cac_lenh
else:
   cac_lenh
```

## IF Lồng
__Cú pháp__
```python
if bieu_thuc1:
   cac_lenh
   if bieu_thuc2:
      cac_lenh
   elif bieu_thuc3:
      cac_lenh
   else
      cac_lenh
elif bieu_thuc4:
   cac_lenh
else:
   cac_lenh
```
